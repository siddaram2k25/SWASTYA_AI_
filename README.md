# SWASTYA_AI ‚Äî Pre-Diagnostic Healthcare Assistant

<img width="170" alt="Icon" src="https://raw.githubusercontent.com/siddaram2k25/SWASTYA_AI_/main/logo.png" align="right" />

**SWASTYA_AI** is an AI-powered pre-diagnostic healthcare assistant designed to help users understand possible health conditions based on their symptoms.  
Using **Naive Bayes ML** and **spaCy NLP**, it extracts symptoms from natural language inputs and predicts the **top most probable diseases**, along with suggested remedies and guidance for further consultation.

---

## üåü Features
- Natural language symptom extraction using **spaCy NLP**
- Predicts **Top 3 most probable diseases**
- Displays associated symptoms for each prediction
- Responsive and simple **HTML / CSS / JavaScript** UI
- Backend powered by **Flask (Python)**
- Works fully offline ‚Äî no external APIs needed

### 1. Clone the Repository


```bash

```
